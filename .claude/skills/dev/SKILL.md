# Start Dev Server

1. Start a Python HTTP server in the background on port 8000:
   ```
   python3 -m http.server 8000
   ```
   Run this in the background so the conversation continues.

2. Open the browser:
   ```
   open http://localhost:8000
   ```

3. Confirm the server is running and the site is open.
